# 3D Model Assets

## seraph.glb (Coming Soon)

The Ascending Seraph 3D model for AR experiences.

### Specifications
- Format: GLB (GLTF Binary)
- Polygon count: Optimized for mobile AR
- Textures: Golden metallic PBR materials
- Animations: Idle float, wing spread, ascend

### Creation Options
1. **Blender** — Free, create from scratch or convert SVG to 3D
2. **Spline** — Free web-based 3D tool
3. **Ready Player Me** — For character-based models
4. **Sketchfab** — Find royalty-free models to customize

### Placeholder Behavior
Until the GLB is added, the AR viewer will show the 2D emblem as a poster image with a message to view in AR when the model is ready.

### Adding the Model
1. Create or commission a 3D Seraph model
2. Export as GLB format
3. Place in this folder as `seraph.glb`
4. The AR experience will automatically use it
